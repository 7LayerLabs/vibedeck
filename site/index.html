The deck

  
Stop alt-tabbing between three AI assistants.

  
Every pane is a real PTY — click in and type like a normal terminal.
    VibeDeck just makes them act like one machine.

  

    

      
      
Broadcast

      
Type in the top bar, hit Enter, and the prompt lands in every pane —
        Claude Code, Codex, Grok, or three Claudes if that's your mood.

    

    

      
      
Real terminals

      
xterm.js + a real PTY per pane. Answer permission prompts, run slash
        commands, drop images — it's the actual CLI, not a wrapper API.

    

    

      
      
Compare + AI judge

      
See every model's answer side by side, crown a winner, and keep a
        running tally — or let a headless judge rule on the round and merge the best ideas.

    

    

      
      
Relay + pipelines

      
Pipe one pane's answer into another as its next prompt — or run
        hands-free chains: plan → build → review across three different models.

    

    

      
      
Live token meter

      
Today's usage and api-rate cost across all three CLIs — per model,
        cache reads included, plan window tracked. Know what the deck is burning.

    

    

      
      
Playbooks + history

      
Starter prompts as plain markdown files, one click to insert.
        Every broadcast saved and searchable.

    

  






  

    
Run your deck

    
Three minutes from clone to cockpit.

    

      

        
STEP 1

        
Install

        
Node + the AI CLIs you already use. No accounts, no cloud — VibeDeck runs on your machine.

        > git clone github.com/7LayerLabs/vibedeck
> npm install && npm start
      

      

        
STEP 2

        
Point it at a project

        
Pick your project folder from the header. Every pane relaunches there — same repo, three brains.

      

      

        
STEP 3

        
Broadcast

        
Type once. Watch three models race. Compare, judge, relay, ship.
